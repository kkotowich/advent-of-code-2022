# advent-of-code-2022
https://adventofcode.com/2022

## Setup
```
cp .env.example .env
```
Add the session_id into the new .env file. This can be found by authenticating with your Advent of Code account.

## Adding a new day
```
mkdir day_1
cp ./day_example/day_example.rb ./day_1/day_1.rb
```

## Running a day
```
ruby ./day_1/day_1.rb
```

## Ruby Version
ruby 2.6.10p210
